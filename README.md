# Next-Paint.io

A Multiplayer Drawing App Created Using Next JS & Websockets.

![next-paint.io](https://user-images.githubusercontent.com/75238302/204462115-a1781f45-b18c-4984-b0a5-b63c02d95e86.png)

## Technologies

### Frontend

-   Next JS v13
-   Typescript
-   TailwindCSS

### Backend

-   Node JS
-   Express JS
-   Typescript
-   Socket.io
